What's the difference?

Performance. FPS. Smoothness. User Experience.

Why?

transform properties are mostly hardware accelerated and can use the devices GPU for rendering processes. This calculation is mostly faster, than using the CPU, because of the BUS between the CPU and the graphics card.

All other properties, especially those for positions, colors, margins or paddings, are calculated through the CPU.

More about this can be found here.

But...

Like everything else, heavy usage of such effects should be avoided. Give the user some nice feedbacks and eye candies. But don't leave him with a truck load of moving pixels. He'll get the same type of eye cancer, such as when using <marquee> or <blink>.
